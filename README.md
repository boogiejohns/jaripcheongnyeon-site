# 자립, 다시 설계하다 — 자립준비청년 공론화 웹사이트

보호종료 후 5년간의 현금성 지원이 '자립 역량 형성기'를 비우고, 그 5년이 절벽처럼 끝나는 구조적 모순을 **스크롤 내러티브**로 설득하고, 주장을 뒷받침하는 사례·연구·기사를 **자료 아카이브**로 모은 한 페이지 사이트.

순수 HTML/CSS/JS. 빌드 단계 없음.

## 구조

```
index.html        스크롤 내러티브(Part A) + 자료 아카이브(Part B)
css/styles.css    다크 테마, 절벽 강조색, reduced-motion 대응
js/main.js        스크롤 등장 애니메이션, 카운트업, 절벽 그래프, 아카이브 필터/검색
resources.json    아카이브 데이터 — 여기에 항목만 추가하면 사이트에 자동 반영
```

## 로컬 실행

`resources.json`을 `fetch`로 불러오므로, 파일을 직접 열지 말고 정적 서버로 실행하세요.

```bash
# 둘 중 아무거나
python -m http.server 8000
npx serve .
```

브라우저에서 http://localhost:8000 열기.

## 자료 추가하기 (이 사이트의 핵심)

`resources.json` 배열에 아래 형식으로 항목을 추가하면 끝. 코드 수정 불필요.

```json
{
  "id": "고유id",
  "title": "제목",
  "summary": "한 줄 요지",
  "tags": ["태그1", "태그2"],
  "link": "https://출처-링크",
  "connection": "우리 주장과의 연결고리 한 줄"
}
```

태그 필터 버튼은 데이터에 등장하는 태그로 자동 생성됩니다.
내러티브의 "자세히 →" 링크와 카드를 연결하려면, 해당 항목의 `id`를 `index.html`의
`data-jump="<id>"` 값과 맞추면 됩니다.

## 배포

정적 사이트이므로 Vercel / GitHub Pages / Netlify 어디든 그대로 올리면 됩니다.

## 원칙

- 화살은 당사자가 아니라 지원의 **설계**를 향한다.
- 인과는 단정하지 않는다(검증되지 않은 가설임을 명시).
- 반례(잘 설계된 현금) 섹션으로 균형을 유지한다.
- 접근성: 시맨틱 마크업, 키보드 내비게이션, `prefers-reduced-motion` 존중.
